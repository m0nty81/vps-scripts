# VPS Deploy Bash

Базовый проект для развертывания на VPS с использованием bash скриптов.

## Структура проекта

```
├── bin/           # Исполняемые скрипты
├── lib/           # Библиотечные функции
├── config/        # Конфигурационные файлы
├── scripts/       # Вспомогательные скрипты
├── tests/         # Тесты
├── logs/          # Логи (игнорируются в git)
├── docs/          # Документация
└── .gitignore
```

## Начало работы

1. Добавьте конфигурационные файлы в папку `config/`
2. Поместите основные скрипты в `bin/`
3. Используйте `lib/` для общих функций
4. Логи будут сохраняться в `logs/`

## Использование

Запуск основных скриптов:

```bash
./bin/deploy.sh
```
